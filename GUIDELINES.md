# Development Guidelines

## 1. Language Standards
- **Code**: Everything must be written in English (variables, functions, comments, component names)
- **UI/Content**: Portuguese-BR for user-facing text
- **Documentation**: Technical docs in English, user docs in Portuguese

## 2. Project Structure
- **Mockup folder**: Reference only, DO NOT modify
- **src folder**: Main development folder - all implementations go here
- **Assets**: Always use the `/assets` folder for images, icons, fonts, and other static files

## 3. Technology Stack
- **Framework**: React Native with TypeScript
